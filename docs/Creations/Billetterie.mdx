# Billetterie

La billetterie TiBillet/Event permet tout type de réservation ou de billetterie et s'inscrit dans tout l'écosystème TiBillet.
Réservez votre place pour un specatcle en monnaie locale. Prenez une demi journée dans un atelier de fablab en monnaie temps, créez des tarifs préférentiels pour vos adhérents, etc ...

- Concert payant avec tarif préférentiel pour les adhérants.
- Concert avec des billets nominatifs ou pas.
- Évènement avec inscription gratuite et nominative ou pas.
- Réservation de table de restaurant non nominative mais nécessitant une validation préalable.
- Évènement gratuit avec une recharge cashless obligatoire.
- Annnonce de réunion, de conférences, etc ...
- BIENTÔT : Réservation a l'heure de salle de réunion, de bureau de co-working, d'ateliers ou de fablab.
- BIENTÔT : Badge NFC pour accès aux évènements, aux salles, aux ateliers, etc ...


## Architecture

TiBillet/Event est pensé en fédération. Une instance peut acceuillir plusieurs lieux.
Cela rend possible le partage d'adhésions, d'abonnements, de membres associatifs.

Cela permet aussi de créer un agenda commun à tout un térritoire.
